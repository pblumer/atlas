Durable BPMN 2.x workflow engine. This is the Milestone S skeleton UI.

A browser BPMN viewer (and later an editor) will render here, reusing bpmn-js with its assets embedded in this binary (see ADR-0011). For now this shell exercises the HTTP API directly. Deployments are held in memory and are lost on restart until durable deployment lands.
